## 3. 配方("seed from geo" 的正确实现)

照搬验证过的 **v13 从零起训 200M 配方**,只换关键三处:
- **种子 = geo 当退火老师**:`--teacher-checkpoint run_063 --anchor-policy-kl-weight 0.35 --anchor-anneal-steps 50000`。geo 在前 5 万步把 207M 拉过新手期,然后**淡出**,让模型靠 d20 数据自己超越。(这不是权重暖启动——207M 与 38.6M 形状不同——而是输出蒸馏,完全合法且安全。)
- **数据 = d20 oracle 管线**:selfplay_runs_stage2_v12 + v126_day3_d4_slice + 三个 v133 verified-blunder 切片(比例 0.70/0.10/0.08/0.06/0.06)+ 人类引导数据。
- **损失**:policy 1.0 / value 0.5 / wdl 1.0 / teacher_q 0.03 / oracle_alpha 0.03,lr 2e-4,warmup 2000。

> 注:今天"窄切片有毒"的教训在这里**不适用** —— 那是 policy-only + 极窄切片 + 不退火地模仿更强老师。这里是 **value+policy 同训 + 全分布 + 退火老师**,正好绕开那个坑。
